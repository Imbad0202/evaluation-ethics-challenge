# 評鑑倫理挑戰 (Accreditation Ethics Challenge)

互動式評鑑倫理教育遊戲，專為高等教育評鑑委員設計。

## 遊戲簡介

玩家扮演評鑑委員，在模擬的評鑑過程中面對各種倫理抉擇。透過情境式學習，培養正確的評鑑倫理觀念。

### 特色功能

- **35 題題庫**：涵蓋評鑑前、中、後各階段的倫理議題
- **5 個關卡**：模擬完整評鑑流程（前置作業 → 實地訪評 → 資料審閱 → 座談訪談 → 報告撰寫）
- **四維倫理評分**：信念堅定、工作紀律、人際溝通、社會責任
- **16 種委員類型**：根據答題結果分析您的委員人格特質
- **動態反饋系統**：學校反彈指數即時反映您的選擇影響

### 響應式設計

支援桌面、平板、手機等各種裝置

### 無障礙支援

- 螢幕閱讀器支援 (ARIA 標籤)
- 鍵盤導航
- 減少動態效果模式
